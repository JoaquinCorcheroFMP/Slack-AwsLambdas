# Slack-AwsLambdas

## To run the project locally
- install lambda-local (npm install lambda-local -g)
- Set the following environment variables:
    - SLACK_VERIFICATION_TOKEN
    - SLACK_ACCESS_TOKEN
    - KEY
    - MONGO_USER
    - MONGO_PWD
